# Práctica IA – Clasificación Car Evaluation

Autor: Miguel Angel Cardona Torrero

## Descripción

Proyecto de clasificación del dataset Car Evaluation usando:

- Árbol de Decisión
- Random Forest

Incluye un menú interactivo para:

- Cargar el dataset
- Análisis exploratorio
- Preprocesamiento
- Entrenamiento y evaluación de modelos
- Curva ROC multiclase
- Comparación de accuracy

## Requisitos

- Python 3.8+
- pandas, numpy, matplotlib, seaborn, scikit-learn

Instalación:
Se creo en un etorno virtual con pyton version 3.11.9
